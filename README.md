# zsh-easy-extraction
zsh-ex is plugin for zsh to easy extract any archive file(.zip .tar .7z) with just one ex command.

### Prerequisites
- unzip
- tar 
- 7-zip

### Installation with [zap](https://github.com/zap-zsh/zap)
```bash
# Put this into your .zshrc
plug "sadikeey/zsh-ex" 
```

### How to use 
```bash
ex test.zip
```
